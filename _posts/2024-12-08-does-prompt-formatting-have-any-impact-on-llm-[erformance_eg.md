# Does Prompt Formatting Have Any Impact on LLM Performance?

## Problem

This paper explores whether the format of prompts (e.g., plain text, Markdown, JSON, YAML) impacts the performance of GPT-based large language models (LLMs). While much research has been conducted on other aspects of prompt engineering (like in-context learning or few-shot example ordering), the effect of prompt structure has received little attention. The authors aim to answer three core questions:

- How sensitive are LLMs to prompt format variations?
- Do LLMs provide consistent responses when different formats are used for identical queries?
- Is there a universally optimal prompt format across models?
